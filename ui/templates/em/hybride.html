{# ── Qu'est-ce qu'une IA Grounded ? ── #}

{{ _('Qu\'est-ce qu\'une IA « Grounded » ?') }}

{{ _('Le terme « Grounded » (ancré) désigne une IA dont les réponses sont fondées sur des données vérifiées, et non sur l\'imagination ou des probabilités linguistiques.')|safe }}

{{ _('La différence fondamentale avec les modèles de langage classiques (ChatGPT, Gemini en mode libre, Claude) est cruciale :') }}

  • {{ _('Un LLM classique génère une réponse qui « semble correcte » à partir de ses connaissances d\'entraînement. Il peut inventer des faits, confondre des données ou produire des réponses plausibles mais fausses — c\'est ce qu\'on appelle l\'hallucination.')|safe }}
  • {{ _('Une IA Grounded comme HYBRIDE interroge d\'abord sa base de données, récupère les faits pertinents, puis formule sa réponse à partir de ces données vérifiées. Pas de place pour l\'invention.')|safe }}

{{ _('L\'analogie simple') }}

ChatGPT = {{ _('un ami qui répond de mémoire. Souvent pertinent, parfois complètement faux, et vous ne pouvez pas vérifier.') }}

HYBRIDE = {{ _('un bibliothécaire qui va chercher dans les archives avant de répondre. Plus lent d\'une seconde, mais toujours factuel.') }}

{{ _('En termes techniques, HYBRIDE utilise une architecture appelée RAG (Retrieval-Augmented Generation) : le modèle « récupère » d\'abord les données pertinentes dans la base, puis « augmente » sa réponse avec ces données vérifiées. Le résultat : zéro hallucination sur les questions EuroMillions.')|safe }}
